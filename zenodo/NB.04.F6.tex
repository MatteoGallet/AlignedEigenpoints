\section{Proposition}\label{proposition}}

    If the five points \(P_1, \dots, P_5\) satisfy \[
\delta_1(P_1, P_2, P_4)=\overline{\delta}_1(P_1, P_2, P_3) =
  \overline{\delta}_1(P_1, P_4, P_5) = 0
\] and if we impose the condition that there is an eigenpoint, say
\(P_6\), aligned with \(P_2\) and \(P_4\), then the eigenpoints satisfy
